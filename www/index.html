Mission

Generic syntax highlighting support for Intelli/J IDEA. Uses simple configuration files to add syntax highlighting for eg Ruby, Python, etc.

In contrast to writing a full-blown IDEA plugin for every language this plugin should have a new syntax available within minutes. In contrast to the XML filetypes extension that IDEA offers, this plugin is more flexible. (But of course less flexible than writing a full-blown language-specific IDEA plugin.. :)

Since version 0.18 a JParsec based approach is used for the lexer configuration. The syntax is specified using a Groovy or JRuby script. The script creates a JParsec lexer which is then adapted to the IntelliJ lexer API.

Usage

After installing the plugin via the IDEA plugin manager, the Tools menu contains two new menu entries. Use 'Init SimpleSyntax' to activate the plugin. This will install some default configuration files and make the Ruby syntax highlighting available for *.rb and *.ruby files.

The configuration files used by the plugin are stored in IDEA's 'config/options' folder. For Windows systems this would be something like 'C:\Documents and Settings\Daniel\.IntelliJIdea60\config\options\SimpleSyntax'. There you can change the existing Ruby configuration or add new configurations for Python, ActionScript, or whatever.. :)

After changing the configuration files, use the 'Reload SimpleSyntax' entry in the Tools menu to reload the configuration files and make your changes available.

Status

  • This plugin is usable, but in an alpha stage.
  • Highlighting fails for multi-line syntax elements. But only while editing. Reopening a file fixes these issues. (Internal info: Lexer state is not handled, yet.)
  • Performance is acceptable for source code sizes up to a few thousand LOC.

Example Configuration

Here's a part of the example configuration included in the current release of the plugin: 


    [SimpleSyntax:V1.1]

    Name: Ruby
    Icon: Ruby.png
    ExampleCode: RubyExampleCode.rb
    Description: Ruby Script

    # Braces Configuration
    Braces.Pairs: (),[]
    Braces.Structural: {}

    # Commenter Configuration
    Comment.Line: #
    Comment.BlockPrefix:
    Comment.BlockSuffix:

    # FileType Configuration
    FileType.Icon: Ruby.png
    FileType.Extensions: rb, ruby
    FileType.DefaultExtension: rb

    # Syntax Definition
    SyntaxDefinition: RubySyntax.rb

    # Element Descriptions
    descriptions[ LINE_COMMENT ] = Line comment
    descriptions[ DOC_COMMENT ] = Documentation
    descriptions[ SPECIAL_QUOTED_STRING ] = Special Quoted String
    (..snip..)
    descriptions[ NUMBER ] = Number

    # Element Default Attributes
    attributes[ LINE_COMMENT ] = #008000
    attributes[ DOC_COMMENT ] = #008000,BOLD
    attributes[ SPECIAL_QUOTED_STRING ] = #008080
    (..snip..)
    attributes[ NUMBER ] = #800080
